# Marché Butembo

Application web de gestion du Marché de Butembo.

## Technologies utilisées

- PHP 8 ou supérieur
- MySQL ou MariaDB
- HTML5
- CSS3
- JavaScript

## Structure

Marche_Butembo/

- index.php
- README.md
- config/
  - database.php
- views/
- models/
  - traitement/
  - select/
- assets/
  - images/
  - uploads/
    - produits/
    - profiles/
- error_pages/
  - 404.php
  - 403.php
  - 500.php

## Installation avec XAMPP

Placez le projet dans :

C:\xampp\htdocs\Marche_Butembo

Créez ensuite la base de données :

marche_butembo

Ouvrez le projet à l'adresse :

http://localhost/Marche_Butembo/
